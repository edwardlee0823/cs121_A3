How to run the code - 
1. Make sure that you are in the correct directory and that you have the necessary data file (i.e. developer folder) inside the same directory.
2. Inside the terminal, type "py m1.py" to run the code (The command may differ based on the machine).
3. Wait until the index is generated; it may take quite a while (estimated: 30-40 minutes).
4. Then, after the index is created, the print statement saying "Enter seach query (or type 'exit' to quit): " will be written on the terminal.
5. You can type any choice of query to search data inside the provided developer folder.
6. If you want to exit, you can simply type "exit" inside the terminal.
